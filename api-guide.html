APIキーの作り方

DifyでAIエージェントを外部から利用するためにはAPIキーが必要です。ここでは、APIキーの作成手順を解説します。

1

ホーム画面にログイン

Difyのプラットフォームにログインします。

2

Settingsを開く

画面左側のナビゲーションメニューから「Settings」をクリックします。

3

API keysを開く

Settingsメニュー内にある「API keys」を選択します。

4

「create new secret key」を押下

新しいAPIキーを作成するために「create new secret key」ボタンをクリックします。

5

名前とプロジェクトを選択し、「create secret key」を押下

APIキーの名前を入力し、関連付けたいプロジェクトを選択した後、「create secret key」ボタンを押して作成を完了します。

APIキーで遊んでみよう

作成したAPIキーを使って、DifyのPlaygroundでAIエージェントの動作を試してみましょう。

1

ホームページを開き、「Playground」を押下

Difyのホームページにアクセスし、上部のナビゲーションバーにある「Playground」をクリックします。

2

「Assistants」を選択

Playground画面で「Assistants」タブを選択し、アシスタント機能を利用するための準備をします。

3

アシスタントの名前とシステムインストラクションを記載

作成するアシスタントの名前と、そのアシスタントにどのような役割や指示を与えるかのシステムインストラクションを入力します。

4

APIモデルを決定

アシスタントが利用するAPIモデル（例: GPT-4oなど）を選択します。

5

動かしてみる

設定が完了したら、Playgroundで実際にアシスタントに質問を投げかけ、その動作を確認します。
